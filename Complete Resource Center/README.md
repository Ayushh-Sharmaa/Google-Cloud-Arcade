## 📚 Google Cloud Arcade – Complete Resource Center

This section provides a curated collection of **official resources, guides, dashboards, and support contacts** to help learners navigate the **Google Cloud Arcade Program** effectively.

Each resource below plays an important role in helping participants:

• Understand the Arcade program structure  
• Complete labs and skill badges  
• Track points and progress  
• Access official guides and dashboards  

---

# 🌐 Official Arcade Platforms

## 🎮 Google Cloud Arcade Website
The **Google Cloud Arcade portal** is the primary platform where all Arcade games, challenges, and learning tracks are hosted.

Participants can:

• Play Arcade Games  
• Complete hands-on labs  
• Earn Arcade points  
• Unlock reward tiers  

🔗 https://go.cloudskillsboost.google/arcade

# 📝 Registration & Participation

## 📄 Arcade Subscription Form (Mandatory)
All participants must submit the **Arcade Subscription Form** at least once per season to ensure their progress is properly tracked.

After submitting the form, you will receive confirmation via email and start receiving **Arcade Insider updates**.

🔗 https://forms.gle/2h6xCvY3sW29pw4p7

---

## 👤 Google Skills Public Profile Settings

Your **Google Skills Public Profile** must be configured correctly before participating.

Required settings:

• Profile visibility set to **Public**  
• Email notifications **Enabled**

# 🎮 Arcade Program Structure

## 🕹️ Arcade Games
Each month the Arcade program releases several learning tracks called **Arcade Games**, each containing hands-on labs and guided cloud tasks.

<p align="center">
<img src="https://github.com/Ayushh-Sharmaa/Arcade-Images/blob/main/Arcade%20Games.png?raw=true" width="100%">
</p>

Game categories may include:

• Arcade Adventure  
• Arcade Voyage  
• Arcade Trail  
• Base Camp  
• Certification Zone  
• Special Games  

---

## ⚡ Arcade Sprint
Arcade Sprint replaces traditional trivia and provides **weekly lab challenges** that allow participants to earn additional points.

<p align="center">
<img src="https://github.com/Ayushh-Sharmaa/Arcade-Images/blob/main/Arcade%20Sprint.png?raw=true" width="100%">
</p>

Each sprint week contains a small set of labs that contribute to the monthly score.

---

## 📊 Arcade Points System
The **Arcade Point System** determines how participants progress through reward tiers.

Points are typically earned by completing:

• Arcade Games  
• Weekly Sprint challenges  
• Skill badges  
• Special games and facilitator bonuses

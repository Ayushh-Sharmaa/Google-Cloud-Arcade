# 🧠 What Are Pre-Assessment Skill Badges?

Pre-Assessment Skill Badges are a **special type of Skill Badge** available on the **Google Cloud Skills platform**.

Unlike regular skill badges that require completing multiple labs, a **Pre-Assessment Skill Badge usually contains only one challenge lab or assessment lab**.

To earn a Pre-Assessment Skill Badge:

• You must complete the **challenge lab / pre-assessment lab**  
• Successfully finish all required tasks in the lab  
• After completion, the **Skill Badge is automatically awarded**

# 🎯 Pre-Assessment Skill Badge Rule

For Pre-Assessment badges:

✔ **1 Lab = 1 Skill Badge**

This makes them different from regular skill badges which contain multiple labs.

---

# 📊 Arcade Point System (Skill Badge)

In the **Google Cloud Arcade Program**, Skill Badges contribute to your total Arcade points.

The standard rule is:

**🏅 2 Skill Badges = 1 Arcade Point**

Example:

| Skill Badges Completed | Points Earned |
|------------------------|--------------|
| 2 Skill Badges | 1 Point |
| 10 Skill Badges | 5 Points |
| 20 Skill Badges | 10 Points |

Pre-Assessment Skill Badges **also count toward this calculation.**

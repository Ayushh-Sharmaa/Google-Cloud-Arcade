# 📚 What Are Lab-Free Courses?

Lab-Free Courses are **theory-based learning modules** available on the **Google Cloud Skills platform** that help learners understand cloud technologies without requiring hands-on lab environments.

These courses typically include:

• Concept explanations  
• Short learning modules  
• Interactive quizzes  
• Knowledge checks  

They help build **foundational understanding before performing hands-on labs or skill badges.**
